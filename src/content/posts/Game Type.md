# 为什么探索太空？  

**1970年5月6日**

亲爱的玛丽·朱昆达修女：

您的来信是我每天收到的众多信件中的一封，但它比其他所有信件都更深刻地触动了我，因为它来自一颗探求真理的心灵和一颗悲悯的心。我将尽力以最好的方式回答您的问题。

首先，我想向您，以及所有像您一样勇敢的修女们，表达我最深的敬佩——你们把一生献给了人类最崇高的事业：帮助那些需要帮助的同胞。

您在信中问道：在地球上还有许多儿童正饿死的时候，我怎么能建议花费数十亿美元去火星旅行？我知道，您并不期待我这样回答：“哦，我之前不知道有儿童饿死，从现在起我会停止一切太空研究，直到人类解决这个问题！”事实上，早在知道去火星在技术上可行之前很久，我就已经知道世界上有饥饿的儿童。然而，我和我的许多朋友一样相信：现在就应该开展登月、最终登陆火星和其他行星的伟大冒险。而且，我甚至相信，从长远来看，这个计划对解决地球上这些严重问题的贡献，将远远超过那些年复一年被讨论却进展极其缓慢的其他援助项目。

在更详细地说明我们的太空计划如何帮助解决地球上的问题之前，我想先讲一个我多年前听到的真实故事，这个故事我一直无法忘记。

大约400年前，德国的一个小镇上住着一位仁慈的伯爵。他把收入的大部分都用来救济镇上的穷人。那是一个饥荒与瘟疫频发的年代，他的善行深受爱戴。一天，伯爵遇到一个奇怪的人。这个人在自己简陋的房子里摆着一张工作台和一个小实验室，整天用玻璃片磨制小透镜，把它们装在管子里，观察极微小的东西。伯爵被这个人用高倍放大镜展示的水滴里游动的微小生物深深吸引。他邀请这个人把实验室搬到城堡里，成为伯爵家的一员，全职改进和完善他的光学装置。

然而，镇上的居民愤怒了。他们认为伯爵在把钱浪费在一个毫无意义的爱好上。“我们正在遭受瘟疫折磨，”他们说，“他却花钱让这个人盯着水滴里的小虫子看！”伯爵却态度坚定。“我已经给了你们我能负担的一切，”他说，“但我还要支持这个人及其工作，因为我知道总有一天会从中产生伟大的东西！”

确实产生了非常伟大的东西。那个人叫安东·范·列文虎克（Anton van Leeuwenhoek），他完善后的显微镜让人类第一次看见了细菌和其他微生物。显微镜比任何其他发明都更推动了医学进步，正是它在很大程度上帮助人类从世界上大多数地区消灭了瘟疫和其他传染病。

那位伯爵用他收入的一小部分支持基础研究，为减轻人类苦难所做的贡献，远远超过他把全部财富直接用于救济瘟疫受害者所能做到的。

我们今天的情况在许多方面与此相似。美国总统每年预算大约2000亿美元。这些钱用于医疗、教育、福利、城市更新、高速公路、交通、对外援助、国防、环境保护、科学、农业以及国内外无数设施。今年，国家预算的约1.6%用于太空探索。太空计划包括阿波罗计划，以及许多较小的项目：空间物理、空间天文学、空间生物学、行星探测、地球资源项目和空间工程。为了实现这些太空支出，年收入1万美元的普通美国纳税人每年为太空计划纳税大约30美元。其余的9970美元用于他自己的生活、娱乐、储蓄、其他税种以及所有日常开支。

您现在可能会问：“为什么不从每个美国纳税人为太空计划缴纳的30美元里拿出5美元、3美元甚至1美元，去救助饥饿的儿童呢？”要回答这个问题，我必须简要说明政府的预算是如何运作的。其他国家的情况也大同小异。国会决定每个部门和每个项目能得到多少资金。如果国会不批准NASA的预算，原本计划给NASA的钱不会自动转去救助饥饿儿童；它根本就不会向纳税人征收，除非其他某个部门的预算相应增加。

您也许会问：我个人是否愿意看到政府采取行动，救助世界各地饥饿的儿童？我的回答是毫不含糊的“是”。事实上，如果我的年度税单因此增加几美元，我一点也不会介意。

我所有的朋友都持同样的看法。然而，仅仅取消太空计划并不能实现这样的救助项目。相反，我甚至相信，通过从事太空计划，我正在为解决地球上的贫困与饥饿等严重问题做出一些贡献。

饥饿问题主要有两个方面：粮食生产和粮食分配。在世界某些地区，粮食生产效率极高，但在许多地区却严重不足。大片土地如果采用高效的水土保持、化肥使用、天气预报、土壤肥力评估、种植规划、作物调查和收获方法，就能大幅增产。而实现这一切最有力的工具正是人造地球卫星。它在高空环绕地球，能在短时间内扫描大片土地，观测和测量作物、土壤、干旱、降雨、积雪等大量指标，并将信息传输到地面站采取相应行动。据估算，即使是一个配备了地球资源传感器的中等规模卫星系统，运行几年后，通过全球农业和林业增产所带来的收益，就能远远超过其全部成本。

把粮食分配给需要的人则完全是另一个问题。困难不在于运输，而在于国际关系。一个小国的统治者可能不愿接受大国的大量粮食援助，只因为他担心随之而来的影响和外国势力。我担心，在国家边界变得不像今天这样壁垒森严之前，高效的饥饿救济不会到来。太空飞行不可能一夜之间创造这个奇迹。然而，太空计划无疑是目前最有希望、最强大的推动力量之一。

请允许我提醒您最近发生的阿波罗13号险情。当宇航员返回舱即将再入大气层的关键时刻，苏联暂停了所有使用阿波罗项目频段的无线电发射，以避免干扰，并将苏联船只部署在太平洋和大西洋，以备紧急救援。如果返回舱落在苏联船只附近，苏联人毫无疑问会像对待自己返回的宇航员一样全力救助。美国人在类似情况下也会这样做。

通过轨道观测提高粮食生产、通过改善国际关系促进粮食分配，只是太空计划深刻影响地球生活的两个例子。我再举两个例子：技术发展和科学知识的生成。

登月航天器对精度和可靠性的极端要求，在工程史上前所未有。满足这些严苛要求的过程，给了我们独特的机会去发现新材料、新方法，发明更好的技术系统，改进制造工艺，延长仪器寿命，甚至发现自然界的新定律。

所有这些新技术同样可以应用于地球。每年，大约1000项来自太空计划的技术创新进入民用领域，带来更好的厨房电器、农机具、缝纫机、收音机、船舶、飞机、天气预报、风暴预警、通信设施、医疗仪器以及日常工具。您可能会问：为什么我们必须先为登月宇航员开发生命支持系统，才能为心脏病患者制造远程传感器？答案很简单：重大技术进步往往不是直奔目标，而是先设定一个极高的挑战目标，它激发创新、点燃想象力，并引发一系列往往比原始目标更重要的连锁反应。

太空飞行毫无疑问正在扮演这样的角色。去火星的旅行本身不会直接成为饥饿者的食物来源，但它将带来如此之多的新技术和新能力，仅这些副产品就很可能远远超过整个项目的成本。

除了新技术，我们还需要源源不断的新基础科学知识，才能改善地球上人类的生活条件。我们需要在物理学、化学、生物学、生理学，尤其是医学领域获得更多知识，以应对威胁人类生命的诸多问题：饥饿、疾病、食品和水污染、环境污染。

我们需要更多选择科学作为职业的年轻人，需要更好地支持那些有才华、有决心从事高水平研究的科学家。必须有足够挑战性的研究目标，必须有充足的经费支持。太空计划以其研究月球、行星、物理学、天文学、生物学和医学的绝佳机会，几乎是理想的催化剂，它能在科学动力、观察自然奇妙现象的机会以及研究所需物质支持之间引发完美的化学反应。

在美国政府所有由国家主导、控制和资助的活动中，太空计划无疑是最显眼、也最受争议的一项，尽管它只占国家总预算的1.6%，占国民生产总值的不到0.33%。作为新技术开发和基础科学研究的催化剂，它是无与伦比的。在这方面，它接过了过去几千年里战争所扮演的悲剧性角色：刺激技术进步、产生科学知识。如果各国用宇宙飞船而非战争火箭来竞争，人类将避免多少苦难！

尽管我们的太空计划看似把我们带离地球，奔向月球、太阳、行星和恒星，但我相信，没有任何一个天体能像地球这样被轨道航天器如此深入地研究。它将成为一个更美好的地球——不仅因为我们把所有新技术和新知识应用于改善生活，更因为我们正在培养对地球、生命和人类的更深刻珍惜。

随信附上的照片是1968年圣诞节阿波罗8号绕月飞行时拍摄的地球景象。在迄今为止太空计划的众多成果中，这张照片或许是最宝贵的。它睁开了我们的眼睛，让我们看到地球是茫茫虚空中最美丽、最珍贵的岛屿，除了这颗行星薄薄的表层，我们无处可去。从未有如此多的人同时意识到地球是多么有限，破坏它的生态平衡将是多么危险。自这张照片首次公布以来，警告人类在我们时代面临的污染、饥饿、贫困、城市化、粮食生产、水资源控制、人口过剩等严峻问题的声音越来越响亮。年轻太空时代的到来，恰好让我们第一次从外部好好看了一眼我们的地球，这绝非偶然。

此致  
敬礼  

恩斯特·斯图林格  
NASA马歇尔太空飞行中心 科学副总监
